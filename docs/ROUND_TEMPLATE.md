# Round Entry Template

Use this template when adding a new row to `/docs/ROUND_HISTORY.md`.

## Required Fields

- Date: `YYYY-MM-DD`
- Round id: `Rxx`
- Branch: `codex/rxx-<scope>`
- Goal: one sentence
- Status: `Completed` or `Blocked`
- Notes: validation summary + key risks

## Example

| 2026-02-12 | R01 | `codex/r01-parser-bootstrap` | Build strict XML parser skeleton and source refs | Completed | Unknown tags detected: 14 (expected for bootstrap); parser tests added |
